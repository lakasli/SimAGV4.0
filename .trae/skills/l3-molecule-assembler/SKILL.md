# 目标
把多个原子组装成一个更大能力单元，但依赖仍保持“伞状”，不产生网状耦合。

# 约束
- 只调用L4原子（及允许的基础库/标准库）
- 不调用任何其他L3分子
- 不承载跨模块流程编排（流程放在L2）

# 工作流
1. 列出所需原子清单（按输入输出串起来）
2. 设计分子对外契约（对L2暴露什么能力）
3. 实现组装逻辑（尽量无分支流程；分支流程交给L2）
4. 输出集成测试用例表（覆盖关键组合路径）

# 输出内容
- 分子模块代码
- 调用到的原子列表（依赖清单）
- 集成测试用例清单
